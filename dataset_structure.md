This repository accompanies the dataset:

"Whole-genome DNA methylation dynamics during brain aging in rare minnow".

Directory description:

01_global_methylation
    Genome-wide methylation summaries.

02_genomic_feature_methylation
    Methylation levels across genomic features.

03_chromosome_methylation
    Chromosome-level methylation profiles.

04_PCA_data
    Methylation matrices used for principal component analysis.

05_DMR_results
    Differentially methylated regions identified using metilene.

metadata
    Sample information used in the analyses.
